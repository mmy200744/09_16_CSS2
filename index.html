여행 추천 프로젝트

사용자의 취향과 선호도에 맞춰 여행지를 추천하는 웹 서비스입니다. React와 Node.js를 사용하여 개발하였으며, 카카오맵 API를 활용하여 지도 서비스를 제공합니다.

재능 매칭 프로젝트

자신의 재능을 공유하고 다른 사람의 재능을 배울 수 있는 플랫폼입니다. Spring Boot와 JPA를 사용하여 개발하였으며, 실시간 채팅 기능을 구현하여 사용자 간의 원활한 소통을 지원합니다.

AWS 인프라 프로젝트

AWS 서비스를 이용하여 확장 가능하고 안정적인 인프라를 구축하는 프로젝트입니다. Terraform을 사용하여 인프라를 코드로 관리하고, Jenkins를 이용하여 CI/CD 파이프라인을 구축했습니다.
